## Community Guidelines

To keep the Discord server helpful and welcoming:

### Be Respectful

- Treat all community members with respect
- Be patient with beginners
- Assume good intent in questions and discussions
- Keep conversations constructive and on-topic
- No harassment, discrimination, or offensive content

### Ask Effective Questions

Help others help you:

<Steps>
  <Step title="Search first">
    Check the FAQ, Wiki, and previous Discord messages before asking.
  </Step>
  
  <Step title="Provide context">
    Include:
    - What you're trying to accomplish
    - What you've already tried
    - Relevant error messages (use code blocks)
    - iSH version and iOS version
  </Step>
  
  <Step title="Use appropriate channels">
    Post in the right channel for your topic (support, development, general, etc.).
  </Step>
  
  <Step title="Be patient">
    Community members help in their free time. Give people time to respond.
  </Step>
</Steps>

### Share Responsibly

- Don't spam or post duplicate messages
- Use threads for long discussions
- Don't share pirated content or copyrighted material
- Keep file sharing relevant and safe
- Use appropriate channels for different topics

### Report Issues Properly

For bugs and crashes:

1. **Discord is for discussion** - Report bugs on [GitHub Issues](https://github.com/ish-app/ish/issues)
2. **Provide details** - See [Contributing](/community/contributing) for what to include
3. **Discuss first** - If you're unsure if something is a bug, ask on Discord first

<Warning>
Don't share crash dumps containing personal information in public channels. Use GitHub issues or DM moderators if needed.
</Warning>
